# Die Besteuerung des Vermögens von Eigentümern

Es gibt einige [schwer reiche Personen](https://www.forbes.com/real-time-billionaires/) auf dieser Welt. Und sie sind dort gelandet, weil [sie weniger Steuern zahlen als du](https://wwnorton.co.uk/books/9781324002727-the-triumph-of-injustice!). 

Eine Massnahme in diesem Spiel besteht also darin, sie zu besteuern. Du entscheidest, wie viel – von 0 % (= du lässt sie so reich, wie sie derzeit sind) bis 80 % (= du nimmst ihnen jedes Jahr 80 % ihres angesammelten Vermögens weg, damit deine Regierung in Bildung, Sicherheit, Straßen usw. investieren kann). Oder, wie immer, alles dazwischen: 5 %, 12 %, 25 % usw.
