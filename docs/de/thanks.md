# Danke

Wir verwenden das Regionalmodell [earth4all](https://earth4all.life/the-science/), das von U. Goluke und P. E. Stoknes entwickelt wurde. Es basiert wiederum auf dem von J. Randers entwickelten [globalen](https://stockholmuniversity.app.box.com/s/uh7fjh52pvh7yx1mqfwqcyxdcvegrodf) Modell. Der Prototyp des Spiels wurde mit [anvil.works](https://anvil.works/) von U. Die Produktionsentwicklung erfolgte mit Hilfe von [Claude Sonnet](https://claude.ai/login) für die Benutzeroberfläche, die Zustandsverwaltung und die VPS-Bereitstellung. Die Übertragung des Modells von [Vensim](https://vensim.com/) nach Python sowie alle Grafiken wurden mit [PyCharm](https://www.jetbrains.com/pycharm/) und [VSCode](https://code.visualstudio.com/) von U Golüke erstellt.
