# Reduzierung von Lebensmittelabfällen

[Wir verschwenden eine Menge Lebensmittel](https://www.wri.org/insights/how-much-food-does-the-world-waste). Mit dieser Massnahme kannst du etwas dagegen tun! Du entscheidest, um wie viel der Anteil an *normalem* Abfall, der bei 30 % liegt, reduziert werden soll. Das heißt: 100 bedeutet, dass du den gesamten Lebensmittelabfall auf null reduzierst – überhaupt kein Abfall mehr! 50 bedeutet, dass der Abfall um 50 % reduziert wird, 0 bedeutet, dass der Abfall wie bisher weiterläuft.
