# Wiederaufforstung

[Der Rückgang der Waldflächen ist schlecht](off-balance-sheet-financing.md), aber man kann als Gegenmaßnahme Bäume pflanzen. Dies wird als [Wiederaufforstung](https://www.earthday.org/reforestation-fact-sheet/) bezeichnet. 

0 bedeutet keine Wiederaufforstung, 1 bedeutet, dass man die Waldfläche um 1 % pro Jahr vergrößert, 3 bedeutet, dass man die Waldfläche um 3 % pro Jahr vergrößert (in 10 Jahren hat man die Waldfläche um etwa **34 %** (!) vergrößert – solange es Land gibt, das wieder aufgeforstet werden kann!)
