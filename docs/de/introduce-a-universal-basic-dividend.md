# Einführung eines universellen Grundeinkommens

Ein [allgemeine Grunddividende](https://earth4all.life/universal-basic-dividend/) entsteht, wenn ein Staat Gemeingüter wie Fischereirechte (siehe z. B. [hier](https://www.sciencedirect.com/science/article/abs/pii/S0964569118300656)), Bergbaurechte (siehe z. B. [hier](https://www.igfmining.org/financial-benefits/global-mining-tax-initiative/)), das Recht zur Nutzung von Funkfrequenzen (siehe z. B. [hier](https://www.ofcom.org.uk/siteassets/resources/documents/consultations/category-2-6-weeks/114665-annual-licence-fees/associated-documents/statement-annual-licence-fees-900-mhz-and-1800-mhz.pdf?v=323615)) usw. besteuert. 

Diese Massnahme legt diese Steuer als Prozentsatz deines BIP fest, d. h. 0 = 0 % des BIP, also nichts; 5 = 5 % des BIP; 3 = 3 % des BIP.  Das gesamte Geld, das du auf diese Weise einnimmst, fließt in die allgemeinen Steuereinnahmen der Regierung. Die Regierung kann dieses zusätzliche Geld dann für [neue Investitionen oder Subventionen für seine Bürger:innen](https://www.economicshelp.org/blog/glossary/difference-between-consumption-and-investment/) verwenden.

Besonders erfolgreiche Beispiele sind der [Alaska Permanent Fund](https://de.wikipedia.org/wiki/Alaska_Permanent_Fund) und der [Staatlicher Pensionsfonds Norwegens](https://de.wikipedia.org/wiki/Staatlicher_Pensionsfonds_(Norwegen)). Kennst du andere?
