# Technologie

Du brauchst eine gute Internetverbindung, denn alle werden gleichzeitig mit dem Internet verbunden sein. 

Jeder moderne Browser auf jedem Gerät (Handy, Tablet, Laptop, PC) funktioniert, aber auf Handys sieht alles etwas klein aus. 

Dem:der Spielleiter:in können ein Projektor und eine Leinwand zur Verfügung gestellt werden, da ihm:ihr die **weltweiten** Ergebnisse angezeigt werden (den Spieler:innen werden nur die Ergebnisse für ihre Region und ihr Ministerium angezeigt). Der:die Spielleiter:in möchte vielleicht darüber sprechen, wie sich einzelne regionale Entscheidungen auf globaler Ebene auswirken.
